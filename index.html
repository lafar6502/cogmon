    Introduction

    CogMon is a real-time reporting software for analyzing time-based data series - like business process performance statistics, KPIs, web site activity, hardware performance statistics and any other parameters that change with time. CogMon offers an extensible, user-customizable web GUI for viewing the reports, building one's own dashboards and constructing new types of reports.

    Requirements

    CogMon is a .Net application. It requires ASP.Net MVC 3, Microsoft.Net 4.0 and a Windows operating system. Data is stored in MongoDB database. There were no attempts to run CogMon on Mono/Linux.

    How it works

    Data is collected by querying various data sources for current values of monitored parameters in regular intervals (e.g. every 5 minutes)

    Why not Cacti

    Cacti is targeted for hardware/network infrastructure monitoring and lacks some features important for non-IT users.

    What kinds of information can be collected

    Any numeric parameter that can be queried for current value can be monitored.
